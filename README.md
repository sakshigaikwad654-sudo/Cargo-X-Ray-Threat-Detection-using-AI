## 📌 Overview

ClearScan AI is an intelligent deep learning-based system designed to automatically detect prohibited or dangerous items in baggage X-ray images using YOLOv8.

---

## 🔍 Features

* Real-time object detection
* Detects weapons (knife, firearm, explosive)
* High accuracy using YOLOv8
* Grad-CAM explainability
* Automated threat alert system

---

## 🧠 Tech Stack

* Python
* YOLOv8 (Ultralytics)
* OpenCV
* NumPy

---

## ⚙️ Working Pipeline

1. X-ray image capture
2. Image preprocessing
3. YOLOv8 detection
4. NMS filtering
5. Threshold check
6. Threat classification
7. Human verification

---

## 📊 Results

* High detection accuracy
* Fast inference (~47 ms)

---

## 🎯 Applications

* Airport baggage screening
* Cargo inspection
* Security surveillance
